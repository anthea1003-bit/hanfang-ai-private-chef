# 漢方私廚網頁原型

手機優先、無外部套件的互動試玩版。

## 開啟方式

可直接用瀏覽器開啟 `index.html`。若瀏覽器對本機檔案有限制，也可在此資料夾執行：

```bash
python3 -m http.server 4187 --bind 127.0.0.1
```

再開啟 `http://127.0.0.1:4187/`。

## 已實作

- 20 種食品級漢方種子食材。
- 20 道固定食譜資料，依已選食材、過敏原與時間回傳前三名。
- 手機照片預覽與候選食材試玩；不會把照片上傳。
- 食譜詳情、逐步陪煮及語音按鈕介面試玩。
- 台北、台中、高雄示範購買點與外部地圖導航。
- 手機底部導覽、桌面雙欄編排與水彩紙張風格。
- 主視覺、頁面背景、20 種食材插畫與 20 道食譜插畫皆由 GPT Image 2 生成；功能按鈕與文字仍保持可存取的網頁元件。
